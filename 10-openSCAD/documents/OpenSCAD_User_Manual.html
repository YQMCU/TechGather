翻译自[OpenSCAD用户手册](https://en.wikibooks.org/wiki/OpenSCAD_User_Manual) --- # 1. 介绍 --- OpenSCAD是用于创建实体3D CAD对象的软件。 它是免费软件，可用于`GNU/Linux`,`Microsoft Windows`和`OS X`系统。 与大多数用于创建3D模型的免费软件（例如知名应用程序[Blender](https://en.wikibooks.org/wiki/Blender)）不同，OpenSCAD不注重3D模型的艺术方面，而注重CAD(计算机辅助设计)方面。因此，当你计划创建机器零件的3D模型时，可能是你正在寻找的程序；但当你更有兴趣创建电脑动画电影或有机生活模型时，可能不是你正在寻找的应用程序。 不同于很多CAD产品，OpenSCAD不是一个交互型建模软件。相反，它更像是一个2D/3D编译器，读取描述对象的程序文件并且从这个文件中渲染成模型。这使得设计师完全控制建模过程，设计师可以轻松地改变建模过程中的任何步骤。这使你可以进行由配置参数定义的设计。 OpenSCAD有两种主要的操作模式，预览和渲染。预览使用3D图形和计算的GPU，相对较快，但是是模型的近似值，并且可以产生伪像；预览使用了[OpenCSG](http://opencsg.org/)和[OpenGL](https://en.wikipedia.org/wiki/OpenGL)。渲染产生精确几何和完全可以 --- # 2. 起步 --- ## 2.1 创建一个简单模型 第一个模型：创建一个 2*3*4 的长方形 ```javascript cube([2,3,4]); ``` 编译并渲染模型 **F5**编译预览 **F6**渲染 ## 2.2 打开已存在例子模型 打开例子程序，或者复制粘贴简单例子到编辑器窗口。 ```javascript difference(){ cube(30,center=true); sphere(20); } translate([0,0,-10]){ cylinder(h=40,r=10); } ``` 然后点击F5按键获得图形预览，或者点击F6获取图形视图。 在预览框架中，获取三种动作： * 拖拽鼠标左键转动视图。同时，底部状态栏中的rotate值相应变化。 * 拖拽鼠标右键平移视图。同时，底部状态栏中的translate值发生相应变化。 * 使用鼠标滚轮放大缩小。 ## 2.3 定位一个对象 我们已经看到如何创建一个简单的长方体。我们下一个任务是意图使用`translate`命令放置完全相同的长方体在已经存在长方体之后。 ```javascript cube([2,3,4]); translate([3,0,0]){ cube([2,3,4]); } ``` `translate`命令之后没有分号。 ## 2.4 改变对象的颜色 我们可以通过给RGB值改变对象的颜色。改变颜色仅仅在预览模式下有效，渲染模式暂时不支持颜色。 ```javascript color([1,0,0]) cube([2,3,4]); translate([3,0,0]) color([0,1,0]) cube([2,3,4]); translate([6,0,0]) color([0,0,1]) cube([2,3,4]); } ```
