# Vue.js是什么 `Vue.js`是一套构建用户界面的**渐进式框架**。 Vue采用自底向上增量开发的设计。 Vue核心库只关注图层，并且非常容易学习，非常容易与其它库或已有项目整合。 另一方面，Vue完全有能力驱动采用[单文件组件](http://cn.vuejs.org/v2/guide/single-file-components.html)和[Vue生态系统支持的库](http://github.com/vuejs/awesome-vue#libraries--plugins)开发的复杂单页应用。 Vue.js目标是通过尽可能简单的API实现**响应式数据绑定**和**组合的视图组件**。 点击[对比其他框架](http://cn.vuejs.org/v2/guide/comparison.html) # 起步 **学习Vue.js需要有HTML、CSS和JavaScript中级前端知识**。 安装Vue.js请看[安装指南](http://cn.vuejs.org/guide/installation.html) 测试可以下载[开发版本](http://vuejs.org/js/vue.js) BootStrap中文网 CDN服务 : [//cdn.bootcss.com/vue/2.1.10/vue.js](//cdn.bootcss.com/vue/2.1.10/vue.js) 在开发版本中**遇到常见错误会给出友好的警告**。 `vue-cli`构建单页面使用，**不推荐**新手直接使用，尤其对Node.js构建工具不够了解更要慎用。 # 声明式渲染 Vue.js的核心是一个允许你采用简洁的模板语法来声明式的将数据渲染进DOM的系统： 示例`first_demo.html`如下： ```html
{{message}}
``` ```javascript var app = new Vue({ el:"#app", data:{ message:"Hello Vue" } }); ``` 返回结果： ```dos Hello Vue ``` 如是，即生成了第一个Vue应用！ 现在数据和DOM已经绑定在一起，所有元素都是**响应式的**。 通过测试可以看出。打开开发者工具的console控制台，输入指令 ```javascript app.message = "It's Cool" ``` 返回结果： ```dos It's cool ``` 除了绑定插入的文本内容，还可以采用下述方式绑定DOM元素属性： 示例`second_demo.html`如下： ```html
Hover your mouse over me for a few seconds to see my dymaically bound title
``` ```javascript var app2 = new Vue({ el:"#app-2", data:{ message:"You loaded this page on "+new Date() } }); ``` 上例中，当鼠标放在div#app-2悬停，会显示You loaded this page on 时间。也就是div元素默认的title属性和message属性绑定在了一起。 此处`v-bind`属性被称为**指令**。指令带有前缀`v-`，以表示它们是Vue.js提供的特殊属性。 `v-bind:x="y"`指令是将这个元素的`x`属性和Vue实例的`y`属性绑定到一起。 再次输入`app2.message = "some new message"`就会再次看到title属性的已经进行了更新。 # 条件与循环 控制切换一个元素的显示： 示例`third_demo.html'如下： ```html

Now you see me

``` ```javascript var app3 = new Vue({ el:"#app-3", data:{ seen:true } }); ``` 返回结果： ```dos Now you see me ``` 如果在控制台设置`app3.seen = false`，div就消失了。 上述例子演示了vue.js不仅可以绑定DOM文本到数据，也可以绑定DOM**结构**到数据。 而且vue.js也提供一个强大的过渡效果系统，可以在vue插入/删除元素时自动应用 [过渡效果](http://cn.vuejs.org/v2/guide/transitions.html) [管理过渡状态](http://cn.vuejs.org/v2/guide/transitioning-state.html) 除了`v-if`还有其他指令，比如`v-for`指令可以绑定数据到数组来渲染一个列表。 ```html
  1. {{todo.text}}
``` ```javascript var app4 = new Vue({ el:"#app-4", data:{ todos:[ {text:'Learn JavaScript'}, {text:'Learn Vue'}, {text:'Build something awesome'} ] } }); ``` 返回结果： ```dos 1.Learn JavaScript 2.Learn Vue 3.Build something awesome ``` 在控制台输入`app4.todos.push({text:Learn Python})`，会发现列表会多一条。 同样，如果`app4.todos.pop()`，列表会清空最后一条。 如是观之，vue.js将DOM文本和Vue数据动态捆绑在了一起，并且可以**像操作JavaScript中的数组一样操作**。
