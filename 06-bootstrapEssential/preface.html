[TOC] --- # Preface # 前言 --- As the number of Internet users on mobile devices is growing every moment,your websites are no longer built only for the desktop machines. The mobile-first philosophy demands the sites to be fully compatible for all available and future mobile devices. Bootstrap allows and easily enables you to design and develop your websites congenial to all devices including various screen readers. 随着互联网用户数量每时每刻的增长，你的网站不再是只为台式机而创建。移动优先的理念要求网站完全兼容所有可用的未来移动设备。Bootstrap允许并且更简单的使你设计开发你的网站适应所有的设备包括各种屏幕阅读器。 --- We have until now developed your websites using all those manual CSS classes and quite a number of various JavaScript libraries.Delivering the desired results and upgrading your websites has traditionally been quite a challenge. Mobile devices coming into the picture has made the task even more diffcult. Bootstrap comes to your rescue here – providing all you need including CSS classes and JavaScript components in a single package. 直到现在，我们使用所有那些手动CSS类和相当多的JavaScript库来开发你的网站。实现期望的结果并升级你的网站一直都是一个相当大的挑战。移动设备进入图片使得任务更加困难。Bootstrap来拯救你——提供所有你需要的包括CSS类和JavaScript组件在一个简单的包中。 --- This book covers all the theoretical and practical aspects of Bootstrap and makes you a proficient web developer for the mobile world. You will be able to download,include, and configure Bootstrap in your web project. You will understand the internal architecture and structure of Bootstrap and get fully familiar with the usage of Bootstrap CSS and components and become apprised of the JavaScript objects offered by Bootstrap. You will also be able to build and compile Bootstrap from the source code, and finally customize and extend to suit your requirements. 本书涵盖了所有Bootstrap的理论和实践，让你成为一个移动领域熟练的网络开发者。你能够下载，包含并在你的网站项目中配置Bootstrap。你会理解Bootstrap的内部架构并且完全熟悉Bootstrap CSS和组件的用法，以及被告知Bootstrap提供的JavaScript对象。你也可以建立和编译Bootstrap源码，最后定制和扩展以满足你的需求。 --- ## What this book covers ## 本书概览 --- Chapter 1, The Evolution of CSS and Bootstrap, introduces you to a brief history of CSS and the birth of CSS3. It introduces the mobile-first philosophy and lays down the fundamental concepts and requirements of Bootstrap. 第1章，CSS和Bootstrap的演变，向你介绍了CSS的简史和CSS3的诞生。它介绍了“移动优先”的理念和涉及到的基本概念与Bootstrap的需求。 --- Chapter 2, Getting Started with Bootstrap, enables you to download and include Bootstrap in your project. You will get to know the files' and folders' structure of different distributions of Bootstrap. You will also start building a sample application using Bootstrap framework. 第2章，Bootstrap起步，使你能够下载和在你的项目中包含Bootstrap。你将了解不同分发版本的文件和文件夹结构。你也将开始使用Bootstrap框架创建一个示例应用。 --- Chapter 3, Creating Responsive Layouts Using Bootstrap CSS, goes through all the major aspects of CSS classes offered by Bootstrap, including basic HTML elements, responsive classes, handling images, Bootstrap grid system, data entry forms, and so on. 第3章，使用Bootstrap CSS创建响应式布局，通晓Bootstrap CSS类的主要概念，包括基本的HTML元素，响应式类，处理图片，Bootstrap栅格系统，数据输入形式，等等。 --- Chapter 4, Packaged Components in Bootstrap, explores all the components that come in-built with the Bootstrap framework. For example, the Glyphicons, navigation bars, toolbars, panels, wells, and pagination. You will use all these components in the sample application to understand the practical usage. 第4章，Bootstrap包组件，探索所有的Bootstrap框架内置组件。例如，字体图标，导航栏，工具栏，面板，well以及分页。你将使用所有这些组件在示例应用中以了解实际用法。 --- Chapter 5, The JavaScript Add-ons in Bootstrap, makes you proficient with all the important JavaScript add-ons that Bootstrap offers. You will be able to use the modal windows, tabs, pills, accordians, tooltips, dropdowns, popovers, alerts, carousels, and so on. 第5章，Bootstrap JavaScript插件，使你精通重要的Bootstrap JavaScript插件。你将能够使用模态框，标签页，pills，手风琴，提示框，下拉菜单，弹出框，警告框，轮播，等等。 --- Chapter 6, Compiling and Building Bootstrap, introduces you to the advanced usage of Bootstrap and how to establish the development environment of Bootstrap, how to compile the Bootstrap source code, and how to build the distributable version of the framework. 第6章，创建和编译Bootstrap，向你介绍Bootstrap的高级用法，如何搭建Bootstrap开发环境，如何编译Bootstrap源码，以及如何创建框架的可分发的版本。 --- Chapter 7, Customizing Bootstrap, enables you to use the development environment and customize Bootstrap's default behavior to suit your own requirements. 第7章，定制Bootstrap，使你能使用开发环境，定制Bootstrap默认行为以满足你自己的需要。 --- Chapter 8, Extending Bootstrap, introduces you to the Bootstrap community marketplace where you will find different resources by which you can extend and broaden the Bootstrap default capabilities and features. For example, you will use Tree-view control and WYSIWYG editors, which are not a part of Bootstrap default offering. Finally, you will be able to create the sample application with almost all the major classes and components of Bootstrap along with the community controls. 第8章，扩展Bootstrap，向你介绍的Bootstrap社区市场，在此你可以发现能扩展和扩大Bootstrap默认功能和特性的不同资源。例如，你将使用树型视图控制和所见即所得的编辑器，这些不是Bootstrap默认提供的功能。最终，你将可以与社区控制一起使用几乎全部的主要Bootstrap类和组件创建示例应用。 --- ## What you need for this book ## 阅读此书你需要什么能力 --- This book is a step-by-step and detailed guide to develop full device-friendly websites. It starts with the basic concepts of CSS, CSS3, and Bootstrap. It includes all the practical know-hows about all major features and functionalities offered by Bootstrap. It also shows you how to work with the Bootstrap main source code and how to customize and extend the framework. 这本书是一本循序渐进的详细引导开发完全设备友好的网站。它从基本的概念开始，如CSS，CSS3和Bootstrap。它包含了所有关于Bootstrap主要特性和功能的实践技巧。它也向你展示了如何使用Bootstrap主源码工作，以及如何定制和扩展Bootstrap框架。 --- An example application is accompanied by every step as an easy-to-follow practical exercise and demonstrates the usage of the features of Bootstrap. 一个应用示例是伴随着每一步作为一个容易学习的实践经验和示范Bootstrap特性的用法。 --- In order to understand and work on this book as a reader, you need to be knowledgeable in HTML, CSS, and JavaScript. Knowledge of HTML5, CSS3, and jQuery will be an added advantage; however, the book is not essentially dependent on those prerequisites. 作为读者，为了理解和使用本书，你需要了解HTML，CSS和JavaScript。虽然了解HTML5，CSS3和jQuery将更有优势，但是这本书本质上不是依赖这些先决条件。 --- ## Who this book is for ## 这本书写给谁 --- Bootstrap Essentials is intended for all those web developers who design and develop the websites and pages using HTML, CSS, and JavaScript—irrespective of whether you have experience on mobile website development or not, this book will guide you to become an adroit Bootstrap web developer. For better understanding of Bootstrap and mobile website development, previous experience of HTML, CSS,and JavaScript will be helpful. Further knowledge in jQuery would be an added advantage. Advanced information about building and customizing Bootstrap are part of the book; thus, readers who want to have their own version of Bootstrap will be able to establish a full-fledged development environment of Bootstrap. Bootstrap精粹适用于所有不论是否有移动网站开发经验的使用HTML，CSS，JavaScript的设计和开发网站的开发者，本书将引导你成为一个熟练的Bootstrap网站开发者。为了更好的理解Bootstrap和移动设备开发，HTML，CSS和JavaScript的经验是有用的。进一步了解jQuery将更有优势。关于创建和定制Bootstrap是本书的一部分内容，因此，想要拥有他们自己版本的Bootstrap版本的读者将可以建立一个完全成熟的Bootstrap开发环境。 ---
