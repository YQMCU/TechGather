Several browsers/drivers are supported (Firefox,Chrome,Internet Explorer,Phantomjs),as well as the Remote protocol. ## 2.2 安装 selenium for Python 安装版本`selenium 3.0.2` window环境下： 使用pip安装 pip install selenium 或者，下载源发布包(例如:selenium-3.0.2.tar.gz)，解压以后，进入文件夹，使用python指令安装安装 python setup.py install ## 2.3 Drivers Selenium requires a driver to interaction with the chosen browser. 也就是说selenium本身需要有真实或者虚拟的浏览器作支持。 而针对不同的浏览器又有着不同drivers 如果selenium选用的driver不被支持会报出错误`selenium.common.exception` 主流的浏览器driver如下： * Chrome:[https://sites.google.com/a/chromium.org/chromedriver/downloads](https://sites.google.com/a/chromium.org/chromedriver/downloads) * Edge:[https://developer.microsoft.com/en-us/microsoft-edge/tools/webdriver/](https://developer.microsoft.com/en-us/microsoft-edge/tools/webdriver/) * Firefox:[https://github.com/mozilla/geckodriver/releases](https://github.com/mozilla/geckodriver/releases) * Safari:[https://webkit.org/blog/6900/webdriver-support-in-safari-10/](https://webkit.org/blog/6900/webdriver-support-in-safari-10/) ## 2.4 入门示例 ### Example 0: * 打开一个**Phantomjs**浏览器 * 通过URL载入页面 示例`phantomjs_load_url.py`如下 ```python # -*- coding = utf-8 -*- # File name :chrome_load_url.py from selenium import webdriver phantom_path = r'D:\2017-02-22\desktop20170222\69xe\node_modules\phantomjs-2.1.1-windows\bin\phantomjs.exe' browser = webdriver.PhantomJS(executable_path=phantom_path) browser.get("http://www.baidu.com/") ``` 使用phantomjs的前提是将`phantomjs.exe`加载在系统环境变量中，或者将路径设置为**executable_path**参数。 ### Example 1: * 打开一个Chrome浏览器 * 载入页面 * 查找`seleniumhq` * 关闭浏览器 示例如下： ```python ```
